Alliance OS Official Builds page

There are many possible combination of Linux Distributions and/or custom installation that are valid for building Alliance OS binaries. In order to ease the work for the Alliance OS builder this page lists all environments where a successfull build was performed.

The Compile Host column display the OS type and version that host the compile/development process, while the Target CPU column shows which is the CPU where Alliance OS build will run to and Tool Name & Version list all required program tools needed to correctly produce a valid Alliance OS build.
